# Quality Attribute Scenario 3 - Geração de IDs em diferentes formatos 

| **Elemento**              | **Declaração**                                                                                                                                                                                                                                                                                 |
|---------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Atributo de Qualidade** | Modificabilidade (subcategoria de Maintainability)                                                                                                                                                                                                                                             |
| **Estímulo**              | O sistema precisa gerar IDs únicos para várias entidades em diferentes formatos, conforme especificações diferentes, para garantir a identificação correta e eficiente em múltiplos contextos de uso.                                                                                          |
| **Fonte do Estímulo**     | Um desenvolvedor ou administrador de sistema necessita alterar o formato de geração de IDs (de Base65 para Hexadecimal ou vice-versa) para atender a requisitos específicos de um novo cliente ou contexto de uso.                                                                             |
| **Ambiente**              | O sistema opera em um ambiente com uma elevada densidade de informação, onde múltiplos serviços precisam gerar e utilizar IDs de forma consistente e sem conflitos, garantindo a integridade dos dados.                                                                                        |
| **Artefato**              | Módulo de geração de IDs (Interface IdGenerator e suas implementações: Base65IdGenerator, HexadecimalIdGenerator), configuração de profiles Spring Boot (application.properties).                                                                                                              |
| **Resposta**              | 	O sistema permite trocar o formato de geração de IDs alterando apenas o profile ativo no ficheiro de configuração (spring.profiles.active), sem necessidade de modificar código-fonte. A mudança é aplicada no próximo restart da aplicação, garantindo consistência em todos os IDs gerados. |
| **Medição da Resposta**   | 100% dos IDs gerados devem ser únicos e atender às especificações do formato                                                                                                                                                                                                                   |
